Detailed Description

Plugin for generating documentation of registered classes and methods.

This plugin generates the documentation of all registered classes and methods of the linked libug4. Depending on command line parameters either HTML or C++ files are generated.

Usage: 

ugshell -call GenerateScriptReferenceDocu\\(\\"regdocu\\", true, false, true, false\\)

Parameters:

  • string outputPath
    path of the output directory for the html, cpp files and completion list
  • bool silent
    don't print verbose progress info, only warnings
  • bool generateHTML
    generates legacy HTML docu
  • bool generateCPP
    generates dummy C++ code from registered functions ready to be parsed by Doxygen
  • bool generateList
    generates completion list
