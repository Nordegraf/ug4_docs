Detailed Description

template<typename TGridFunc>
class ug::Electromagnetism::EddyCurrentReBofEUserData< TGridFunc >

UserData based class for the computation of the real part of the magnetic field \( \mathbf{B} = \frac{i}{\omega} \mathbf{rot} \, \mathbf{E} = - \frac{1}{\omega} \mathbf{rot} \, \mathrm{Re} \, \mathbf{E} + i \frac{1}{\omega} \mathbf{rot} \, \mathrm{Im} \, \mathbf{E} \), i.e. the vector field \( \mathrm{Re} \, \mathbf{B} = - \frac{1}{\omega} \mathbf{rot} \, \mathrm{Im} \, \mathbf{E} \).
