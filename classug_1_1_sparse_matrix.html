Detailed Description

template<typename TValueType>
class ug::SparseMatrix< TValueType >

sparse matrix for big, variable sparse matrices.

SparseMatrix matrix is stored independent row-wise When doing discretisation, use the add set and get methods for dealing with submatrices of A. For other things you can use the row iterators or operator()-methods.

See also
matrixrow, CreateAsMultiplyOf
